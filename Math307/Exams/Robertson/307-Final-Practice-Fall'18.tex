\question[20]

Answer true or false to the following statements:
\vskip.2in
If $V$ is a vector space of dimension $n$, then $V$ is an $n \times n$ matrix.

\vfill
If $0$ is an eigenvalue of $A$, then $A$ is not invertible.   
\vfill
If $V$ is a vector space of dimension $n$, then any set of $n$ vectors must be linearly independent.

\vfill
If $\det(A) \neq 0$ then $A$ is invertible.
\vfill
If $A$ is an $n \times n$ invertible matrix, then the rows of $A$ are linerarly independent
\vfill
If $v_1$ and $v_2$ are linearly independent then there exists nonzero constants, $c_1$ and $c_2$, such that $c_1 v_1 +c_2 v_2 = 0$. 
\vfill
If $\lambda$ is an eigenvalue of $A$ then $\det(\lambda I - A)=0$. 
\vfill
If $V$ is a vector space of dimension $n$, then any collection of $n$ vectors forms a basis for $V$. 
\vfill
If $A$ is an $n \times n$ matrix with $n$ distinct eigenvalues, then $A$ is diagonalizable. 
\vfill
If $A$ is an $n \times n$ matrix with $n$ distinct eigenvalues, then $A$ is invertible. 

\vfill
If $A$ is an $n \times n$ matrix what is invertible then, then $A$ is diagonalizable. 

\vfill
If $A$ is an $n \times n$ matrix what is diagonalizable then, then $A$ is invertible.
